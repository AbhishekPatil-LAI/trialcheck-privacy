Privacy Policy for TrialCheck

Last Updated: January 2025

1. Overview

TrialCheck is a test application designed for learning and testing Facebook Marketing API functionality. This privacy policy explains how we handle information when you use our application.

2. Information We Collect

Our application is designed for testing purposes and has minimal data collection:

  • Facebook Login Information: When you log in through Facebook, we receive basic profile information (name, email) as permitted by Facebook's API
  • Marketing Data: We access Facebook Ads Manager data only for testing API connections and creating test campaigns
  • Usage Data: Basic application usage for debugging and improvement purposes

3. How We Use Information

We use collected information solely for:

  • Testing Facebook Marketing API functionality
  • Creating test ad campaigns (always in PAUSED status)
  • Debugging and improving application performance
  • Educational and learning purposes

4. Information Sharing

We do NOT:

  • Sell your personal information to third parties
  • Share your data with advertisers
  • Use your information for commercial advertising
  • Store sensitive financial or payment information

5. Data Security

We implement appropriate security measures to protect your information:

  • Secure API connections using HTTPS
  • Limited data access based on necessity
  • Regular security reviews and updates
  • No storage of sensitive personal data beyond testing requirements

6. Data Retention

Since this is a test application:

  • Test data may be deleted regularly for cleanup
  • We retain minimal information only as needed for functionality
  • You can request data deletion at any time

7. Facebook Integration

Our app integrates with Facebook's Marketing API:

  • We follow Facebook's Platform Policies
  • Your Facebook data remains subject to Facebook's own Privacy Policy
  • We only request minimum necessary permissions
  • All ad campaigns created are in PAUSED status by default

8. Children's Privacy

Our application is not intended for children under 13 years of age. We do not knowingly collect personal information from children under 13.

9. Your Rights

You have the right to:

  • Access your personal information
  • Request correction of inaccurate data
  • Request deletion of your data
  • Withdraw consent for data processing
  • Disconnect the app from your Facebook account

10. Changes to Privacy Policy

We may update this privacy policy occasionally. We will notify users of any significant changes through the application or via email.

11. Legal Compliance

This application complies with:

  • Facebook Platform Policies
  • General Data Protection Regulation (GDPR)
  • California Consumer Privacy Act (CCPA)
  • Other applicable privacy laws

12. Testing and Development

Important notes about our testing application:

  • This app is primarily for educational and testing purposes
  • No real advertising campaigns will be launched without explicit user action
  • All test ads are created in PAUSED status to prevent spending
  • The app may be updated frequently for testing new features
